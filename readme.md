Отвечает на вопросы относящиеся к стажировке Cdekstart, не галлюцинирует, говорит информацию только по заданным данным. Находящимся в папке "data". 
Главная механика поиска: Сначала ищет совподающие слова пользователя в базе данных, потом записывает файл в запрос к ЛЛМ и просит отвечать по нему и только по нему.
