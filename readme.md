## LLMRagBot
Чат-бот для консультаций по программе международной стажировки "CdekStart" на основе RAG и LangGraph.

### Как запустить:
1. git clone https://github.com/shever45/LLM_RAG_BOT
2. cd LLM_RAG_BOT
3. docker-compose up --build
4. Перейти на http://localhost:8000/docs
5. Писать в эндпоинт "Chat" в message:'string'

### Какие LLM поддерживает:
Я пробовал только на Ollama(local),
Но также он может поддерижвать как OpenAi так и совместимые с ним(DeepSeek, Groq, Together)
На них я не тестировал, поскольку либо платные, либо закрыты в нашем регионе.

Чтобы изменить LLM:
1. Зайти в файл llm.py
2. Пролистать до функции get_llm
3. выбрать DEFAULT_PROVIDER = (local / openai / compatible)
4. Заполнить сопутствующие поля
   
### Что выполняет:
Отвечает на вопросы относящиеся к стажировке Cdekstart, не галлюцинирует, говорит информацию только по заданным данным. Находящимся в папке "data". 
Главная механика поиска: Сначала ищет совподающие слова пользователя в базе данных, потом записывает файл в запрос к ЛЛМ и просит отвечать по нему и только по нему.
При вопросе о стипендии, задает уточняющий вопрос, про страну. Выдает ответ соответствующий файлам базы данных.
